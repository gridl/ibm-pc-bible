INT 21h AH=11H Find first file using FCB (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 11h
    DS:DX = segment:offset of file control block (FCB)

Output:
    Return: AL = 0 if found file or 0xFF if file not found

Action:

    Find first file using FCB.

Notes:

"?" wildcard recognized in >= DOS 2.1
"*" recognized in >= DOS 3.x

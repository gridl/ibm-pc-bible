INT 21h AH=15H Sequential write using FCB (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 15h
    DS:DX = segment:offset of file control block (FCB)

Output:
    Return: AL = 0 (successful write)
                 1 (disk full or read-only)
                 2 (data transfer area too small)

Action:

    Sequential write using FCB.

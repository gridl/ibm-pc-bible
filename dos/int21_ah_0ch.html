INT 21h AH=0CH Clear input buffer and call a function (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 0Ch
    AL = 01, 06, 07, 08, or 0A

Output:
    Return: AL = clear input buffer and call function specified in AL 
    
Action:

    Clear input buffer and call a function.

Notes:

See also:

    int21 AH=01, AH=06, AH=07, AH=08, AH=0A

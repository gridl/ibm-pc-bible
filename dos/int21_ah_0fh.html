INT 21h AH=0FH Open file using FCB (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 0Fh
    DS:DX = segment:offset of file control block (FCB)

Output:
    DL = new default disk drive number which corresponds to drive letter
    Return: AL = 0 if file opened or 0xFF if unable to open

Action:

    Open file using FCB.

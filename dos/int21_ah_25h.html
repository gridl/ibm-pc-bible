INT 21h AH=25H Set interrupt vector (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 25h
    AL = interrupt number
    DS:DX = pointer to interrupt handler

Output:
    Return: not applicable

Action:
    Set interrupt vector.

INT 21h AH=23H Get File Size using FCB (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 23h
    DS:DX = segment:offset of file control block (FCB)

Output:
    Return: AL = 0 (if successful)
	       = FF (if file not found)

Action:
    Get File Size using FCB.

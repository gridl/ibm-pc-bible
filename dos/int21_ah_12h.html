INT 21h AH=12H Find next matching file using FCB (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 12h
    DS:DX = segment:offset of file control block (FCB)

Output:
    Return: AL = 0 if found file or 0xFF if file not found

Action:

    Find next matching file using FCB.

Notes:

"?" wildcard recognized in >= DOS 2.1
"*" recognized in >= DOS 3.x

This function used after a related call to AH=11H or AH=12H

See also:

Int21/AH=11H, Int21/AH=12H

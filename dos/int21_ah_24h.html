INT 21h AH=24H Set relative record field in FCB (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 24h
    DS:DX = segment:offset of file control block (FCB)

Output:
    Return: not applicable

Action:
    Set relative record field in FCB.
    
    Modifies opened FCB for random operation.
    Sets FCB random record field to current sequential block
	  and record fields. (see source)

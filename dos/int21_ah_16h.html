INT 21h AH=16H Create or truncate file using FCB (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 16h
    DS:DX = segment:offset of file control block (FCB)

Output:
    Return: AL = 0 (successful)
                 FF (unsuccessful)

Action:

    Create or truncate file using FCB.

INT 21h AH=22H Random write using FCB (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 22h
    DS:DX = segment:offset of file control block (FCB)

Output:
    Return: AL = 0 (read successful)
	       = 1 if EOF (no data read)
	       = 2 if DTA is too small

Action:
    Random write using FCB.

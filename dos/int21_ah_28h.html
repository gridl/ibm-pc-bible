INT 21h AH=28H Random block write using FCB (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 28h
    CX = number of records to read
    DS:DX = segment:offset of file control block (FCB)

Output:
    Return: AL = 0 (successful write)
                 1 (diskette full or read-only)
                 2 (data transfer area too small)
	    CX = number of records written

Action:
    Random block write using FCB.

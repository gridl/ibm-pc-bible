INT 21h AH=27H Random block read using FCB (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 27h
    CX = number of records to read
    DS:DX = segment:offset of file control block (FCB)

Output:
    Return: AL = 0 (successful read)
                 1 (end of file)
                 2 (data transfer area too small)
                 3 (partial record or EOF)
	         CX = number of records read

Action:
    Random block read using FCB.

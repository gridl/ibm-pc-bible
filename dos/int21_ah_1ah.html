INT 21h AH=1AH Set disk transfer address (DTA) (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 1ah
    DS:DX = pointer to disk transfer address (DTA)

Output:
    Return: not applicable

Action:

    Set disk transfer address (DTA).

Notes:

    DTA cannot overlap 64k boundary.

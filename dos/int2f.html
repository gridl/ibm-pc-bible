All INT 2Fh functions, ordered by function call number (value of AH on entry).
Some functions have a subfunction (AL). If any other registers control the
function on entry, they are listed as well.

INT 2Fh carries many functions, from many parts of the system and from many
other utilities, as well as Windows.

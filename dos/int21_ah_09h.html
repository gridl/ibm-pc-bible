INT 21h AH=09H Display String (DOS 1.0+)
────────────────────────────────────────

Input:
    AH = 09h
    DS:DX = FAR 16:16 pointer to '$'-terminated strings

Output (DOS behavior):
    AL = '$' (despite official docs)

Action:

    Print the string at DS:DX until the '$' character, which defines the end of
    the string.

Notes:

    DOS checks for CTRL+C and CTRL+Break keys and executes INT 23h if read.

    Standard output is normally the screen. Under DOS 2.0 and higher, standard
    output can be redirected.

See also:

    INT 21H AH=02H WRITE CHARACTER TO STANDARD OUTPUT

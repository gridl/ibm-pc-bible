INT 21h AH=13H Delete file using FCB (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 13h
    DS:DX = segment:offset of file control block (FCB)

Output:
    Return: AL = 0 if file deleted or 0xFF if file not found

Action:

    Delete file using FCB.

Notes:

"?" wildcard recognized in >= DOS 2.1
"*" recognized in >= DOS 3.x

INT 21h AH=04H Auxiliary Output (DOS 1.0+)
──────────────────────────────────────────

Input:
    AH = 04h
    DL = Character to write

Output:
    None

Action:

    Sends the character in DL to standard aux.

Notes:

    DOS checks for CTRL+C and CTRL+Break keys, and executes INT 23h if pressed.

    Normally standard aux is the first serial port. Under DOS 2.0 and higher,
    standard aux may be redirected.

    If standard aux is busy, this function will wait until it can send
    the character.

See also:

    INT 21H AH=03H Auxiliary Input
    INT 14H AH=01H WRITE CHARACTER TO PORT

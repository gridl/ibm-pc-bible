INT 21h AH=21H Random read using FCB (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 21h
    DS:DX = segment:offset of file control block (FCB)

Output:
    Return: AL = 0 (read successful)
	       = 1 if EOF (no data read)
	       = 2 if DTA is too small
	       = 3 if EOF (partial record read)

Action:
    Random read using FCB.
